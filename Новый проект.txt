1 открыть в VS Code (от имени администратора)
  открыть папку с проектом
2 npm i gulp -D (установка локально в проект)
3 в терминале - " npm i " (установка всех пакетов)

4 запуск в режиме разработчика - " npm run dev " (в процессе работы)
5 запуск в режиме продакшена - " npm run build " (завершение готового проекта)

6 "склеивает" svg картинки (из папки svgicons) в одну, после можно каждую отдельно "вызывать" - " npm run svgSprive " (не проверял)
7 создание zip - " npm run zip "
8 выгрузка на ftp " npm run deploy " (нужно заполнить config/ftp.js и в файле path.js заполнить имя папки ftp (стоит `test` по умолчанию)) (не проверял)
